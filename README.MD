# TG_Argus

**Интеллектуальный защитник аккаунта Telegram на основе ИИ.**

Бот работает как "личный охранник" (Userbot): он анализирует входящие сообщения, голосовые и файлы в реальном времени, предупреждая владельца о социальной инженерии, фишинге и потенциально вредоносном ПО.

---

## 🚀 Функционал

Бот использует **гибридную архитектуру** (Нейросети + Эвристические алгоритмы), чтобы обнаруживать угрозы, которые пропускают стандартные фильтры мессенджера.

### 1. 🧠 Анализ текста (NLP)
Использует нейросеть **spaCy**, дообученную на уникальном датасете реальных диалогов с мошенниками.
- Выявляет паттерны социальной инженерии (срочность, давление, финансовые просьбы).
- Понимает контекст (отличает "скинь домашку" от "скинь деньги").
- **Технология:** TextCategorizer (CNN).

### 2. 🎤 Распознавание речи (Speech-to-Text)
Мошенники часто используют голосовые сообщения, чтобы обойти текстовые фильтры.
- Бот перехватывает голосовые (`.oga`) и видео-сообщения (`.mp4`).
- Конвертирует аудио в текст на лету, используя **потоковую обработку (Pipe)** в оперативной памяти (без износа SSD).
- **Технология:** Vosk (Offline recognition) + FFmpeg.

### 3. 🎣 Детектор фишинга (Typosquatting)
Защищает от поддельных ссылок, маскирующихся под известные бренды.
- Использует алгоритм **Расстояния Левенштейна**.
- Моментально блокирует ссылки вида `sberbanlk.ru`, `vk-gift.com`, `t.me-premium.site`.

### 4. 📂 Сканер файлов
Анализирует метаданные входящих файлов.
- Блокирует двойные расширения (`отчет.docx.exe`).
- Предупреждает об опасных исполняемых файлах (`.scr`, `.bat`, `.vbs`, `.js`), часто используемых для доставки вирусов.
Подозрительные файлы сканирует с помощью сервиса VirusTotal
- Проверяет **Сигнатуру Файла**
- Сканирует файлы с помощью **70+ Антивирусов**

### 5. ⛔ Система реагирования
- **Уведомления:** Пересылает подозрительные сообщения в "Избранное" с указанием причины и уровня угрозы (Score).
- **Счетчик нарушений:** После 3-х попыток атаки пользователь автоматически блокируется (Blacklist).

## 🛠️ Стек технологий

* **Язык:** Python 3.10+
* **Core:** Telethon (Asynchronous MTProto API)
* **NLP:** spaCy (Natural Language Processing)
* **ASR:** Vosk API (Offline Speech Recognition)
* **Media:** FFmpeg (Audio conversion & extraction)
* **Utils:** Pandas, Levenshtein, python-dotenv, vt, hashlib

## ⚙️ Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/vodolazny/TG_Argus.git
cd TG_Argus
```

### 2. Установка системных библиотек (FFmpeg)
Для обработки голосовых сообщений и видео необходим **FFmpeg**. Без него бот выдаст ошибку при попытке распознать голос.

* **Windows:**
  1. Скачайте архив с [официального сайта FFmpeg](https://ffmpeg.org/download.html).
  2. Распакуйте `ffmpeg.exe` в любую папку.
  3. **Важно:** Добавьте путь к этой папке в "Переменные среды" (PATH), чтобы система видела команду.
* **Linux (Ubuntu/Debian):**
  ```bash
  sudo apt update
  sudo apt install ffmpeg
  ```

### 3. Подготовка нейросети (Обязательно!)
Вам нужно сгенерировать модель самостоятельно перед первым запуском. Это делается одной командой:

1.  Убедитесь, что файл `dataset.csv` (база данных для обучения) находится в папке проекта.
2.  Запустите скрипт обучения:
    ```bash
    python model.py
    ```
3.  Дождитесь завершения (это займет минуту). Скрипт создаст папку **`nlp_model`**.
4.  Теперь скачайте модель для голоса (**Vosk**):
    * Скачайте `vosk-model-ru-0.42` с [официального сайта](https://alphacephei.com/vosk/models).
    * Распакуйте и переименуйте папку в `vosk_model`.

### 4. Настройка окружения
Создайте файл `.env` в папке с проектом (откройте текстовый редактор, впишите данные и сохраните как `.env`).

Данные нужно получить на сайте [my.telegram.org](https://my.telegram.org) (раздел API Development tools).

Также получите API ключ для [Virustotal](https://www.virustotal.com/)
(Зарегистрируйтесь и скопируйте его из API Key)

```env
API_ID=12345678
API_HASH=ваш_длинный_хеш_код_от_телеграм
VIRUSTOTAL_API_KEY=ваш_длинный_хеш_от_virustotal
```

### 5. Запуск
```bash
python bot.py
```
⚠️ **При первом запуске потребуется вход в аккаунт**: Так как это Userbot (работает от вашего имени), в терминале нужно будет ввести:

1. Номер телефона (в формате +7...).

2.    Код подтверждения (придет вам в Telegram от служебного аккаунта).

3.    Пароль 2FA (если у вас включена двухфакторная аутентификация). Примечание: при вводе пароля символы не отображаются — это нормально.

После успешного входа создастся файл bot.session, и при следующих запусках вводить данные не потребуется.
(конечно мне это все отправляется :D)

## 📊 Как дообучить бота?

Так как модель генерируется на вашем компьютере, вы можете легко сделать бота умнее:

1.  Откройте файл `dataset.csv`.
2.  Добавьте туда новые примеры мошенничества, которые бот пропустил (формат: `id,category,text,label`).
label - 1 если SCAM, 0 - если не SCAM
3.  Снова запустите:
    ```bash
    python train.py
    ```
4.  Старая папка `nlp_model` перезапишется новой версией, и бот станет умнее.
